# Movie Recommendation System

This is an AI-based Movie Recommendation System built using Python, NLP, Scikit-learn, and Streamlit.

## Features
- Recommend similar movies
- Content-based filtering
- Interactive UI using Streamlit

## Technologies Used
- Python
- Pandas
- NumPy
- NLTK
- Scikit-learn
- Streamlit

## Run Project

```bash
streamlit run app.py
```
